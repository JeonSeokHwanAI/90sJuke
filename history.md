# 프로젝트 고도화 작업기: AI와 함께한 개발 경험

이 글은 '90s Juke' 프로젝트를 고도화하면서 겪은 문제들과, 이를 해결하기 위해 AI 에이전트(Antigravity)를 어떻게 활용했는지 기록한 사례 연구(Case Study)입니다.

## 1. [랜덤] 추천 목록 고도화와 AI 활용
- **프롬프트 핵심:**
    > "메인 화면에 매번 새로운 노래 5곡을 추천하는 기능을 원함. 단, 계절(겨울)에 맞는 노래가 먼저 나와야 함."
- **개발 경험 (Developer Experience):**
    단순한 랜덤 로직은 금방 만들 수 있지만, **"계절감을 살리면서도 데이터를 풍성하게 채우고, 곡끼리 자연스럽게 연결하는 것"**은 손이 많이 가는 작업이었습니다. 여기서 Antigravity의 진가가 발휘되었습니다.

- **상세 작업 내용 및 AI 성과:**
    1.  **알고리즘 구현:** '겨울', '눈' 태그 우선 + 빈자리 랜덤 채우기 + 최종 섞기 로직을 단숨에 구현했습니다.
    2.  **화면 오류 수정:** 데이터 누락으로 인한 에러를 감지하고 방어 코드를 추가했습니다.
    3.  **데이터 채우기:** 비어있던 10곡의 데이터(가사, 설명)를 채워 넣는 반복 작업은 AI에게 맡겨 순식간에 해결했습니다.
    4.  **★ 추천 연결 (Antigravity 활용의 백미):**
        > *"추천 꼬리 연결은 Antigravity를 활용해서 자동으로 수정하는 게 너무 편했습니다."*
        
        기존에는 곡들의 분위기를 일일이 파악해서 손으로 연결해야 했지만, Antigravity에게 **"분위기에 맞게 자연스럽게 꼬리물기 해줘"**라고 요청하자 전체 15곡의 맥락을 분석해 자동으로 추천 그래프를 재설계해주었습니다.
        이후 노래를 추가할 때도 복잡한 설정 없이 **"이 노래 구성해줘"**라고만 하면, 알아서 기존 그래프에 자연스럽게 녹아들게 처리되는 경험이 매우 인상적이었습니다.
    5.  **UI 조정:** 글자 크기를 20px로 확대하고 비율을 조정하여 사용자가 원하는 아날로그 감성을 정확히 캐치했습니다.

## 2. [검색] 기능 추가 (Search)
- **프롬프트 핵심:**
    > "기존의 디자인과 '랜덤 추천' 로직은 유지하되, 사용자가 원할 때 원하는 곡을 찾을 수 있게 만들어줘. (제목, 가수, 태그, 한줄평 검색)"
- **구현:**
    - 종이 서류 느낌의 **검색바(Search Bar)**를 리스트 상단 우측에 배치했습니다.
    - 제목, 가수, 태그(예: #겨울), 한줄평 내용을 모두 검색할 수 있습니다.
    - 검색어가 없을 땐 '랜덤 추천'이, 검색어가 있을 땐 '검색 결과'가 나옵니다.

## 3. [운세] 뽑기 (Omikuji) 통합
- **프롬프트 핵심:**
    > "클라이언트의 요청으로 '운세 뽑기' 기능을 추가하되, 기존의 '자동 추천' 기능과 충돌하지 않게 사용자 선택형으로 만들고 싶어." -> "카드 리스트의 빈 공간(6번째 칸)에 자연스럽게 녹아들게 수정."
- **구현:**
    - **운세 카드:** 다른 노래 카드와 똑같은 디자인으로 6번째 자리에 배치했습니다.
    - **인터랙션:** 카드를 클릭하면 흔들리는 애니메이션과 함께 '대길/중길' 운세와 '오늘의 행운 곡'이 그 자리에서 바로 나타납니다.
    - 검색을 할 때는 검색 결과에 방해되지 않도록 운세 카드가 사라집니다.

## 4. [하트] 실시간 좋아요 추가
- **프롬프트 핵심:**
    > "Vercel KV(Upstash) 설정을 완료하고 환경변수까지 연결했어. 이제 '실시간 글로벌 좋아요(Like)' 기능을 구현해줘. 낙관적 UI(Optimistic UI)와 튀어 오르는(Bounce) 애니메이션 적용."
- **구현:**
    - **실시간 연동:** Vercel KV(데이터베이스)를 연결하여 전 세계 사용자가 누른 좋아요 수가 실시간으로 공유됩니다.
    - **UX 강화:** 하트를 누르는 즉시 숫자가 올라가고 통통 튀는 애니메이션이 나옵니다.
    - 노래 카드뿐만 아니라 '운세 결과 카드'에서도 바로 좋아요를 누를 수 있습니다.
